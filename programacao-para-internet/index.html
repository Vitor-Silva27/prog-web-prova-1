Programação para internet

A disciplina de programação para internet tem como objetivo capacitar os alunos a desenvolver páginas, sites e portais estáticos e dinâmicos, além de interfaces de aplicações.

Os alunos aprenderão a utilizar as principais linguagens de programação para web, como HTML, CSS, JavaScript. ao final do curso, os alunos estarão aptos a projetar, implementar e manter aplicações web de alta qualidade e desempenho..

Ementa da disciplina

  • Introdução aos conceitos de Internet
  • Linguagens de Marcação
  • Fundamentos de folhas de estilo
  • Princípios de Design responsivo
  • Princípios de Javascript e suas principais bibliotecas
  • Conceitos Básicos sobre Aplicações Cliente/Servidor
